# Colorizr

## Installation

Set your local Gemfile as:

``gem "colorizr", :git => 'https://github.com/skamkar/survivr_1.git'``

## Usage

- Load gem

``require 'colorizr'``

- Option #1: Enumerate sample colors

``String.sample_colors``

- Option #2: Apply font color (from list of sample colors)

``puts "John".red``

``puts "Paul".green``

``puts "George".blue``

``puts "Ringo".yellow``
